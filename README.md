# Título: Aplicativo de Receitas: Listagem Inicial

O que o aplicativo faz:

Este aplicativo de receitas conecta-se a uma base de dados online (API) para buscar e exibir receitas. Na tela principal, o usuário pode ver uma lista de receitas de Frutos do Mar. Cada item da lista mostra o nome da receita e uma imagem de pré-visualização.
